# Donezo - Task Management Dashboard

A modern, responsive, and fully functional **Task Management Dashboard** built with React. This application provides secure authentication, a clean dashboard interface, and seamless integration with a REST API to manage and visualize task-related data.

##  Authentication Flow

* User logs in using email and password
* Server returns a JWT token
* Token is stored securely in local storage
* Protected routes check authentication status
* Unauthorized users are redirected to login

---

##  API Integration

The application connects to a REST API to:

* Authenticate users
* Fetch dashboard data
* Display tasks and related information
